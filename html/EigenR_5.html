Eigen_cos R Documentation

Matrix cosine

Description

Matrix cosine of a real or complex square matrix.

Usage

Eigen_cos(M)

Arguments

M

a square matrix, real or complex

Value

The matrix cosine of M.

Examples

library(EigenR)
M <- toeplitz(c(1,2,3))
cosM <- Eigen_cos(M) 
sinM <- Eigen_sin(M)
cosM %*% cosM + sinM %*% sinM # identity matrix
