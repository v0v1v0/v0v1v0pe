ellipse R Documentation

Description

Plotting coordinates for an ellipse

Usage

ellipse(h, k, a, b = a, phi = 0, n = 200L)

Arguments

h

x coordinates
k

y coordinates
a

semimajor axis
b

semiminor axis
phi

rotation
n

number of plotting points

Value

A list of matrices of coordinates for the ellipses.
