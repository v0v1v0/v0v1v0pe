null.omit R Documentation

Remove 'NULL' Entries from List

Description

Function removes all items that are NULL or empty from a list or other object.

Usage

null.omit(x)

Arguments

x

object to be checked

Value

x without NULL entries

Examples

l <- list(a=2, b=NULL, c = 3)
length(l) == 3

k <- null.omit(l)
length(k) == 2
