info_image R Documentation

Image information (Informacoes sobre as imagens).

Description

This function returns information about the image (Esta funcao retorna informacoes sobre a imagem).

Usage

info_image(im)

Arguments

im

:Object containing an image (Objeto contendo a uma imagem).

Value

Importa uma imagem.

Author(s)

Alcinei Mistico Azevedo (Instituto de ciencias agrarias da UFMG)

See Also

crop_image , edit_image

Examples

#Carregar imagem de exemplo
im=read_image(example_image(1),plot=TRUE)
info_image(im)
