stream_from_files R Documentation

Generates a two dimensional data stream from data files in a given folder.

Description

Generates a two dimensional data stream from data files in a given folder.

Usage

stream_from_files(folder)

Arguments

folder

The folder with the data files.

See Also

gen_stream.
