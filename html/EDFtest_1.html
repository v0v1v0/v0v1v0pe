EDFtest: Goodness-of-fit Tests based on Empirical Distribution Function

Description

EDFtest provides software for the calculation of goodness-of-fit test statistics and their P-values. The three statistics computed are the Empirical Distribution function statistics called Cram√©r-von Mises, Anderson-Darling, and Watson statistic.
