irank R Documentation

Integer Ranks

Description

Compute the number of elements less or equal the elements in a given vector.

Usage

irank(x, ox = NULL)

Arguments

x

a numeric vector.

ox

order(x), optionally (for efficiency in case order(x) is already known).

Value

A vector of integers.

Examples

x <- rnorm(10)
irank(x)
rank(x)
x <- c(1,2,3,3,0)
irank(x)
rank(x)
