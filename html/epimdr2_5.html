chabaudi R Documentation

Daily measures of malaria infected mice.

Description

Daily data on laboratory mice infected with various strains of Plasmodium chabaudi

Usage

chabaudi

Format

A data frame with 1300 rows and 11 variables:

Line

line number

Day

day of infection

Box

Cage number

Mouse

Mouse identifier

Treatment

Plasmodium strain

Ind2

Unique mouse identifier

Weight

Mouse weight

Glucose

Blood glucose level

RBC

Red blood cell count

Sample

Sample number

Para

Parasite count
