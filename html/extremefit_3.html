Biweight.kernel R Documentation

Biweight kernel function

Description

Biweight kernel function.

Usage

Biweight.kernel(x)

Arguments

x

a vector.

Details

Biweight kernel:

K(x) = 15/16 ( 1 - x^2 )^2 (abs(x)<=1)

We recommend a critical value of 7 for this kernel function.

Examples

plot(function(x) Biweight.kernel(x),-2, 2,
main = " Biweight kernel ")
