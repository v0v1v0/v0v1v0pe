applyNumCheck R Documentation

Apply recodes according to a numerical check data.frame.

Usage

applyNumCheck(GADSdat, numCheck)

Arguments

GADSdat

A GADSdat object.
numCheck

A data.frame as created by createNumCheck.

Details
