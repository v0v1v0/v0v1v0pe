hcauchy R Documentation

Log Hazard Function for a Cauchy Process

Description

Log Hazard Function for a Cauchy Process

Usage

hcauchy(y, m, s)

Arguments

y

Vector of times.
m

Location parameter.
s

Dispersion parameter.

Value

hcauchy returns the log hazard function for a Cauchy process with the given parameter values.

Author(s)

J.K. Lindsey

Examples
