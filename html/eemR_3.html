Usage

Arguments

data

A list containing "file", "x", "em", "ex".

Value

An object of class eem containing:

  • sample The file name of the eem.

  • x A matrix with fluorescence values.

  • em Emission vector of wavelengths.

  • ex Excitation vector of wavelengths.
