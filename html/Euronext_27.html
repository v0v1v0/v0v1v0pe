EN_Ticker_Performance R Documentation

Get Ticker Historical Performance

Description

This function retrieves the historical performance data of a company listed on the Euronext exchange based on its ticker symbol. It returns a table with details such as the highest and lowest stock prices, date of the highest and lowest prices, and other relevant information.

Usage

EN_Ticker_Performance(ticker, escape = FALSE, stock_type = "Eq_Ind")

Arguments

ticker

A character string representing the company's ticker symbol.
escape

Boolean, either TRUE or FALSE. If escape is True, it means you're providing the DNA (ISIN-Market identifier) directly. Giving T to escape is helpful to avoid time-consuming operations; otherwise, F means you need to provide the Ticker symbol, name, or ISIN and the type of market to which it belongs.
stock_type

The type of the ticker: 'Eq_Ind' for Stocks and Indexes, 'Fund' or "F" for Fund tickers, 'Bond' or "B" for Bond tickers, and 'Etfs' or "E" for EFTs.

Value

A data frame containing historical performance data of the specified company. If the ticker is not found, it returns the message "Ticker not found."

See Also

EN_GetISIN to retrieve the ISIN for a given ticker.

Examples


library(httr)
library(httr2)
library(rvest)
library(jsonlite)
library(stringr)

dt1 = EN_Ticker_Performance("XS1195347478-ETLX", escape = TRUE)

print(dt2[, c(1:6,8,12)])

# Get Performance of AEX All-Share Index
