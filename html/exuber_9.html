autoplot.ds_radf R Documentation

Plotting a ds_radf object

Description

Takes a ds_radf object and returns a ggplot2 object, with a geom_segment() layer.

Usage

## S3 method for class 'ds_radf'
autoplot(object, trunc = TRUE, ...)

Arguments

object

An object of class ds_radf. The output of datestamp()
trunc

Whether to remove the period of the minimum window from the plot (default = TRUE).
...

Further arguments passed to methods. Not used.

Value

A ggplot2::ggplot()

Examples



sim_data_wdate %>%
  radf() %>%
  datestamp() %>%
  autoplot()

# Change the colour manually
sim_data_wdate %>%
  radf() %>%
  datestamp() %>%
  autoplot() +
  ggplot2::scale_colour_manual(values = rep("black", 4))
