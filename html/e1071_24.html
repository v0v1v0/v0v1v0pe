element R Documentation

Extract Elements of an Array

Description

Returns the element of x specified by i.

Usage

element(x, i)

Arguments

x

Array of arbitrary dimensionality.
i

Vector of the same length as x has dimension.

Author(s)

Friedrich Leisch

See Also

Extract

Examples

x <- array(1:20, dim=c(2,5,2))
element(x, c(1,4,2))
