%.% R Documentation

Get the attribute of an object

Description

Get the attribute of an object

Usage

obj %.% attrname

Arguments

obj

The object to get the attribute from.
attrname

The name of the attribute to get.

Value

The attribute of the object.

Examples

obj <- list(a = 1, b = 2)
attributes(obj)$test="hello"
`%.%`(obj, "test")
