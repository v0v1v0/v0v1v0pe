as_ts R Documentation

Time Series

Description

Converts a ts_data object into a time series object (ts).

Usage

as_ts(ts)

Arguments

ts

ts_data object

Value

A ts object.

Examples

# Time series from linear trend
ts <- ts_data(12, trend.coeff= c(sample(0:10, 1), sample(1+(1:10)/20, 1)))
as_ts(ts)
