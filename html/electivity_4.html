ivlev_electivity R Documentation

Ivlev's electivity, E

Description

Bounded between -1.0 (avoidance), 0 (random feeding), and +1.0 (preference).

Usage

ivlev_electivity(r, p)

Arguments

r

(Numeric) Resource utilisation.
p

(Numeric) Resource availability.

Value

A numeric vector.
