e_get_data R Documentation

Get data

Description

Get data passed to e_charts.

Usage

e_get_data(e)

Arguments

e

An echarts4r object as returned by e_charts or a proxy as returned by echarts4rProxy.

Value

A list of data.frames, one for each group.

Examples

echart <- cars |>
  e_charts(speed) |>
  e_scatter(dist) |>
  e_lm(dist ~ speed)

echart

e_get_data(echart)[[1]]
