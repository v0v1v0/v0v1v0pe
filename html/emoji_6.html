emoji_count R Documentation

Count the number of emojis in a string

Description

Vectorised over string

Usage

emoji_count(string)

Arguments

string

Input vector

Value

An integer vector

See Also

stringr::str_count()

Examples

string <- paste(c(letters[1:4], emoji_name[1:6]), collapse = " ")

emoji_count(string)

emoji_count(emoji_name[1:6])
