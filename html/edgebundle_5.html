Convert edges

Description

Usage

## Default S3 method:
convert_edges(object, coords)

## S3 method for class 'igraph'
convert_edges(object, coords)

## S3 method for class 'network'
convert_edges(object, coords)

## S3 method for class 'tbl_graph'
convert_edges(object, coords)

Arguments

object

graph object
coords

coordinates of vertices

Value

data frame of edges with coordinates
