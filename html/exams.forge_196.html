mcval R Documentation

Most Common Value

Description

Computes all modes (most common value).

Usage

mcval(x, ...)

## Default S3 method:
mcval(x, ...)

## S3 method for class 'histogram'
mcval(x, exact = FALSE, ...)

compute_modes(x, ...)

mcv(x, ...)

Arguments

x

data object
...

further arguments
exact

logical: either compute the exact mode or use class mids (default: FALSE)

Value

A vector of modes.

Examples

x <- sample(1:5, 15, replace=TRUE)
mcval(x)
