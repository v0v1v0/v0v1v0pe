Eigen_isInvertible R Documentation

Check invertibility

Description

Checks whether a matrix is invertible.

Usage

Eigen_isInvertible(M)

Arguments

M

a matrix, real or complex

Value

A Boolean value indicating whether M is invertible.

Examples

set.seed(666L)
M <- matrix(rpois(25L, 1), 5L, 5L)
Eigen_isInvertible(M)
