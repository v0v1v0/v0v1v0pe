e_zoom R Documentation

Zoom

Description

Zoom on a region.

Usage

e_zoom(e, ..., btn = NULL)

Arguments

e

An echarts4r object as returned by e_charts or a proxy as returned by echarts4rProxy.
...

Any options, see official documentation
btn

A e_button id.

Examples

cars |>
  e_charts(dist) |>
  e_scatter(speed) |>
  e_datazoom() |>
  e_zoom(
    dataZoomIndex = 0,
    start = 20,
    end = 40,
    btn = "BUTTON"
  ) |>
  e_button("BUTTON", "Zoom in")
