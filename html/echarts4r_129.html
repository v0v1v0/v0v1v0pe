e_labels R Documentation

Format labels

Description

Format labels

Usage

e_labels(e, show = TRUE, position = "top", ...)

Arguments

e

An echarts4r object as returned by e_charts or a proxy as returned by echarts4rProxy.
show

Set to TRUE to show the labels.
position

Position of labels, see official documentation for the full list of options.
...

Any other options see documentation for other options.

Examples

mtcars |>
  e_chart(wt) |>
  e_scatter(qsec, cyl) |>
  e_labels(fontSize = 9)

mtcars |>
  group_by(cyl) |>
  e_chart(wt) |>
  e_scatter(qsec, mpg) |>
  e_labels(fontSize = 9)
