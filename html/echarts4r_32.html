e_axis_labels R Documentation

Axis Labels

Description

Convenience function to add axis labels.

Usage

e_axis_labels(e, x = "", y = "")

Arguments

e

An echarts4r object as returned by e_charts or a proxy as returned by echarts4rProxy.
x, y

Labels of axes.

Examples

cars |>
  e_charts(speed) |>
  e_scatter(dist) |>
  e_axis_labels(
    x = "speed",
    y = "distance"
  )
