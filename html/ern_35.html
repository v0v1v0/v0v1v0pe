plot_dist R Documentation

Plot a distribution

Description

Plot a distribution

Usage

plot_dist(d)

Arguments

d

List that defines the distribution (as returned by def_dist_incubation_period() for example)

Value

A ggplot object.

Examples

# Define a `ern` distribution:
gi  = ern::def_dist(
  dist     = "gamma",
  mean     = 6.84,
  mean_sd  = 0.7486,
  shape    = 2.39,
  shape_sd = 0.3573,
  max      = 15
  )

# Plot can be customized like any `ggplot` object:
g = plot_dist(gi) + ggplot2::labs(subtitle = 'your subtitle')
plot(g)
