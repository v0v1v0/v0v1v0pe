Description

Generate a lookup table

Usage

get_lookup(data_list)

Arguments

data_list

a list containing at least 1 dataframe

Value

a dataframe summarizing column names and labels

Examples

x = edc_example()
x$.lookup=NULL
lk = get_lookup(x)
lk
lk %>% tidyr::unnest(c(names, labels))
