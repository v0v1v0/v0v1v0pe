S&P500

Description

Log-returns of S&P500 Index from 05-01-1960 untill 16-10-1987.

Format

returns

a numeric vector

Details

Log-returns of S&P500 Index from 05-01-1960 untill 16-10-1987.

Examples

data(S_P500)
