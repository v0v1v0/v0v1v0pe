EQ-5D-3L Reverse Crosswalk data

Description

Reverse Crosswalk index value table to calculate EQ-5D-5L indices from EQ-5D-3L data for England, Germany, Netherlands and USA. Table uses the values of van Hout et al from the EuroQol analysis tools webpage.

Usage

RCW

Format

An object of class data.frame with 243 rows and 4 columns.

Source

Reverse crosswalk datasets
