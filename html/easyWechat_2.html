WechatRobot

Description

WechatRobot

Usage

WechatRobot(
  key,
  msg = "",
  qyapi = "https://qyapi.weixin.qq.com/cgi-bin/webhook/send?key=",
  msgtype = "text"
)

Arguments

key

The key for the WeChat Work robot.
msg

Text message.
qyapi

WeChat Work API.
msgtype

Type of text message being sent.

Value

response of the request

Examples

WechatRobot(key = "79bdbb23-7f7e-4fb6-b669-97f7e744755e", msg = "SaveRds Success!")
