schwefel_func R Documentation

Schwefel Function

Description

Implementation of n-dimensional Schwefel function.

Usage

schwefel_func(x)

Arguments

x

numeric or complex vector.

Details

On an n-dimensional domain it is defined by

\[f(\vec{x}) = \sum_{i=1}^{n} \left[ -x_{i}\sin(\sqrt{|x_{i}|}) \right],\]

and is usually evaluated on \(x_{i} \in [ -500, 500 ]\), for all \(i=1,...,n\). The function has one global minimum at \(f(\vec{x}) = -418.9829n\) for \(x_{i}=420.9687\) for all \(i=1,...,n\).

Value

The value of the function.

References

Schwefel H (1981). Numerical optimization of computer models. John Wiley & Sons, Inc.
