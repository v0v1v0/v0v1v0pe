des R Documentation

Desription of a data frame or a variable

Description

Description of a data frame or a variable or wildcard for variable names

Usage

des(dataFrame)

Arguments

dataFrame

a data frame

Details

The variable names will be listed with class and the description of each variable

Examples

data(Oswego)
.data <- Oswego
des(.data)
