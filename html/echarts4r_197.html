e_restore R Documentation

Restore Toolbox

Description

Restore Toolbox.

Usage

e_restore(e, btn = NULL)

Arguments

e

An echarts4r object as returned by e_charts or a proxy as returned by echarts4rProxy.
btn

A e_button id.

Examples

cars |>
  e_charts(speed) |>
  e_scatter(dist) |>
  e_datazoom() |>
  e_restore("btn") |>
  e_button("btn", "Reset")
