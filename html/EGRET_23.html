decimalDate R Documentation

decimalDate

Description

Create a decimal date or date/time from a vector.

Usage

decimalDate(rawData)

Arguments

rawData

vector of dates or dateTimes.

Examples

dateTime <- c('1984-02-28 13:56',
              '1984-03-01 00:00',
              '1986-03-01 00:00',
              '1986-10-15 00:00')
decimalDate(dateTime)

dateTime <- c('1984-02-28', 
              '1984-03-01',
              '1986-03-01',
              '1986-10-15')
decimalDate(dateTime)
