pollutants R Documentation

Dataset containing information about the available pollutants

Description

This dataset belongs to the European Environment Agency institution. Contains short information about the available pollutants.

Usage

pollutants

Format

A data frame with 650 rows and 4 variables:

Code

Unique pollutant code identifier

Notation

pollutant's short name

Label

pollutant's full name

RecommendedUnit

Measurement unit, recommended for the pollutant

Source

http://dd.eionet.europa.eu/vocabulary/aq/pollutant/view
