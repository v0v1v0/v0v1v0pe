catif R Documentation

Condition cat

Description

Calls cat if cond==TRUE.

Usage

catif(cond, ...)

condition_cat(cond, ...)

Arguments

cond

logical: condition, if true then cat is called, otherwise not
...

further parameters

Value

Invisibly cond.

Examples

catif(TRUE, "PDF")      # Should appear
catif(FALSE, "Moodle")  # Should not appear
