read_data_package R Documentation

Read data package

Description

Read data package

Usage

read_data_package(packageId, ore = FALSE, env = "production")

Arguments

packageId

(character) Data package identifier
ore

(logical) Return an OAI-ORE compliant resource map in RDF-XML format
env

(character) Repository environment. Can be: "production", "staging", or "development".

Value

(character or xml_document) A resource map with reference URLs to each of the metadata, data, and quality report resources that comprise the packageId.
