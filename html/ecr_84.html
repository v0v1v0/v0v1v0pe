mutInsertion R Documentation

Insertion mutator.

Description

The Insertion mutation operator selects a position random and inserts it at a random position.

Usage

mutInsertion(ind)

Arguments

ind

[integer]
Permutation of integers, i.e., vector of integer values.

Value

[integer]
