q_matrix R Documentation

Create a Q Matrix Object

Description

Provides a way to create an object as a "q_matrix".

Usage

q_matrix(x)

Arguments

x

Either a data.frame or matrix.

Value

A q_matrix object.

See Also

as_q_matrix()

Examples

# Q matrix values
x = matrix(c(1, 0, 0, 1), nrow = 2)

# Q matrix wrapper
q_mat = q_matrix(x)

# Data Frame encoding of Q
q_df = data.frame(
   k1 = c(1, 0),
   k2 = c(0, 1)
)

# Create a Q matrix
q_mat = q_matrix(q_df)
