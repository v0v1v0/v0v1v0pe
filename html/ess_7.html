as_adj_mat R Documentation

Converts an adjacency list to an adjacency matrix

Description

Converts an adjacency list to an adjacency matrix

Usage

as_adj_mat(adj)

Arguments

adj

Adjacency list

Value

An adjacency matrix

Examples

adj <- list(a = c("b", "d"), b = c("a", "c", "d"), c = c("b", "d"), d = c("a", "c", "b"))
as_adj_mat(adj)
