msk.macrophytes.2017.doc.typology R Documentation

Function to write documentation of probabilities of macrophyte river types at a given site.

Description

Function to write documentation of probabilities of macrophyte river types at a given site.

Usage

msk.macrophytes.2017.doc.typology(res,row.no)

Arguments

res

results as provided by one of the functions msk.macrophytes.2017.calc.types or msk.macrophytes.2017.read.compile.evaluate.

row.no

row index of site to be plotted.
