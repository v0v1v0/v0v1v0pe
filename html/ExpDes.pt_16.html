ex6 R Documentation

Dados ficticios 1

Description

Dados simulados de uma normal com media zero e variancia 1 para um experimento em esquema de fatorial triplo.

Usage

data(ex6)

Format

Um data frame com 24 observacoes nas seguintes 5 variaveis.

fatorA

Vetor numerico.

fatorB

Vetor numerico.

fatorC

Vetor numerico.

rep

Vetor numerico.

resp

Vetor numerico.
