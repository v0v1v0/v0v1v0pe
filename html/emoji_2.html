clock R Documentation

emoji version of time

Description

emoji version of time

Usage

clock(time)

Arguments

time

a POSIXct object

Details

This function is vectorized.

Value

Character vector of emojis showing the closest time.

Examples

times <- as.POSIXct("2021-09-17 14:33:21 PDT") + seq(1:30) * 3500
clock(times)
