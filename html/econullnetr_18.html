WelshStreams.prey R Documentation

Macroinvertebrate abundance to accompany part of the food web from upland streams in south Wales, UK

Description

The abundance of 16 potential macroinvertebrate prey taxa to accompany the food web data in WelshStreams.

Usage

WelshStreams.prey

Format

A data frame with 6 rows and 17 variables. Each row represents the abundance of individual macroinvertebrate taxa in each of the six streams. Column 1 contains the stream codes, whilst columns 2â€“17 represent the abundance of each macroinvertebrate taxon summed across three 1-minute kick-samples in each stream.
