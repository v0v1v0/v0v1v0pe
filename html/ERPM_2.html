Function to calculate the number of partitions with groups of sizes between smin and smax

Description

Function to calculate the number of partitions with groups of sizes between smin and smax

Usage

Bell_constraints(n, smin, smax)

Arguments

n

number of nodes
smin

minimum group size possible in the partition
smax

minimum group size possible in the partition

Value

a numeric

Examples

n <- 6
size_min <- 2
size_max <- 4
Bell_constraints(n,size_min,size_max)
