inbounds R Documentation

Clip value to a given range

Description

Clip x to the interval [l, u]. This is useful to enforce box constraints.

Usage

inbounds(x, l, u)

Arguments

x

Value to clip.
l

Lower limit.
u

Upper limit.

Value

l if x < l, u if x > u else x.
