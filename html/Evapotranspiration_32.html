Radiation R Documentation

Radiation Component for Morton Models (Internal)

Description

Estimating the radiation component for using Morton models to estimate evapotranspiration.

Value

This function returns a object named variables which contains variable values for implementing Morton CRAE and CRWE formulations

Author(s)

Danlu Guo
