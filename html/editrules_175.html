Usage

x[i, j, ...]

## S3 method for class 'editlist'
x[i, j, ...]

Arguments

x

an object of class editmatrix or editarray
i

the row index in the edit matrix (numeric, logical or rowname)
j

...

arguments to be passed to other methods. Currently ignored.
