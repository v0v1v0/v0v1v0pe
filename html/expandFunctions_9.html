Description

Convert vector x into a matrix X_{ij} = {x_i}^j

Usage

ePow(x, colParamVector)

Arguments

x

Data vector.
colParamVector

Vector of column powers.

Value

A matrix X of size length(x) X length(colParamVector)

X_{ij} = {x_i}^j

Examples

x <- 1:6
ePow(x,0:2)
