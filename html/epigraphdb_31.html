xqtl_single_snp_mr R Documentation

Single SNP QTL MR evidence

Description

GET /xqtl/single-snp-mr

Usage

xqtl_single_snp_mr(
  exposure_gene = NULL,
  outcome_trait = NULL,
  snp = NULL,
  qtl_type = c("eQTL", "pQTL"),
  pval_threshold = 1e-05,
  mode = c("table", "raw")
)

Arguments

exposure_gene

Name of the exposure gene
outcome_trait

Name of the outcome trait
snp

SNP rsid
qtl_type

"eQTL" or "pQTL"
pval_threshold

P-value threshold
mode

If mode = "table", returns a data frame (a tibble as per tidyverse convention). If mode = "raw", returns a raw response from EpiGraphDB API with minimal parsing done by httr.

Value

Data from GEET /xqtl/single-snp-mr

Examples

## Not run: 
xqtl_single_snp_mr(outcome_trait = "Coronary heart disease")

## End(Not run)
