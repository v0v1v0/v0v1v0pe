xt R Documentation

Simulate deterministic dynamics of a single species

Description

Helper function for symdyn, used to simulate dynamics between disturbance events.

Usage

xt(t0, t1, B0, r)

Arguments

t0

initial time step
t1

desired time step
B0

initial abundance
r

relative growth rate

Value

predicted value of x at time t1
