Log Hazard Function for a Poisson Process

Description

Log Hazard Function for a Poisson Process

Usage

hexp(y, rate)

Arguments

y

Vector of times.
rate

Vector of rates.

Value

hexp returns the log hazard function for a Poisson process with the given parameter value.
