roc R Documentation

roc

Description

Calculates ROC and AUC

Usage

roc(fitted, actual)

Arguments

fitted

Vector with values between 0 and 1
actual

Vector with two levels

Examples

ezplot:::roc(runif(1), sample(c(TRUE, FALSE), 1, replace = TRUE))
ezplot:::roc(runif(3), sample(c(TRUE, FALSE), 3, replace = TRUE))
