e_axis_stagger R Documentation

Stagger Axis Labels

Description

Stagger axis labels.

Usage

e_axis_stagger(e)

Arguments

e

An echarts4r object as returned by e_charts or a proxy as returned by echarts4rProxy.

Examples

df <- data.frame(
  x = c("a very long label", "Another long label"),
  y = 1:2
)

df |>
  e_charts(x, width = 150) |>
  e_bar(y) |>
  e_axis_stagger()
