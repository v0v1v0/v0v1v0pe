parties R Documentation

Extract sender and recipient(s)

Description

Extract sender and recipient(s)

Usage

parties(msg)

Arguments

msg

A message object.

Value

A tibble.

Examples

msg <- envelope() %>%
  from("Gerald <gerald@gmail.com>") %>%
  to(c("bob@gmail.com", "alice@yahoo.com")) %>%
  cc("Craig     < craig@gmail.com>") %>%
  bcc("  Erin   <erin@yahoo.co.uk    >")

parties(msg)
